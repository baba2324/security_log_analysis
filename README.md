# AI Security: Network Intrusion Detection

## 1. Project Overview
- **Objective:** Detect malicious network attacks (DDoS, Probe) using AI.
- **Dataset:** NSL-KDD (Network Security Benchmark).
- **Solution:** Re-purposed the industrial anomaly detection model to cybersecurity.

## 2. Key Result
- **Detection Rate:** **90%+** (Classified Normal vs. Attacks)
- **Insight:** Proved that anomaly detection logic is effective for security.
